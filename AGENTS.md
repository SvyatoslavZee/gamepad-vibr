# AGENTS.md

## Project rules for Codex

- Work only inside this repository.
- Before editing, inspect the relevant files and explain the plan.
- Prefer small, minimal changes.
- Do not rewrite architecture unless explicitly asked.
- Do not touch secrets, tokens, .env files, API keys, or credentials.
- Do not delete files unless explicitly asked.
- If more than 3-5 files need to be changed, explain why before editing.
- After changes, summarize:
  - what was changed
  - which files were changed
  - how to test it
  - possible risks
- Use Russian when explaining changes to the user.
